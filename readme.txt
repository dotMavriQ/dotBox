== Description ==
dotBox is a lightweight WordPress block theme inspired by the Gruvbox color scheme.
It is accessibility-ready and optimized for fast performance.

== Installation ==
1. Download and extract the zip file.
2. Upload the `dotbox` folder to `/wp-content/themes/`
3. Activate the theme in **Appearance > Themes**.
